📧 Gmail via IMAP — 5-Minute Setup

The most straightforward method to get up and running using the Himalaya or Email skill. Uses IMAP + an App Password — no OAuth flow required.

  • 1
    Enable IMAP in Gmail

    In your Gmail settings (web), go to Forwarding and POP/IMAP and select Enable IMAP. Save changes.

  • 2
    Generate an App Password

    Go to your Google Account → Security. Ensure 2-Step Verification is on, then search for App passwords. Create one named KrustyKlaw and copy the 16-character code.

  • 3
    Connect in KrustyKlaw

    In your KrustyKlaw interface (Telegram, Slack, etc.), run the connection command for your skill:

    /connect email  or  /email setup

    Provide your Gmail address and the App Password (not your regular password) when prompted.

  • 4
    Save your credentials below

    Record your Gmail configuration here so KrustyKlaw can confirm the integration is active and auto-populate future commands.

💬 Slack Channel Bot

Wire KrustyKlaw as a Slack bot application. Requires a Slack workspace where you have permission to install apps.

  • 1
    Create a Slack App

    Go to api.slack.com/apps and create a new app from scratch. Give it the name KrustyKlaw.

  • 2
    Configure Bot Token Scopes

    Under OAuth & Permissions, add scopes: channels:history, chat:write, app_mentions:read.

  • 3
    Install & Copy Token

    Install the app to your workspace and copy the Bot User OAuth Token starting with xoxb-.

  • 4
    Run the Slack channel

    Set channel = "slack" in your KrustyKlaw config and paste the bot token. Invite the bot to a channel and mention it to start chatting.

📱 Telegram Assistant

Deploy KrustyKlaw as a Telegram bot for instant mobile AI access. Setup takes under 2 minutes.

  • 1
    Create a Bot via BotFather

    Open Telegram and message @BotFather. Send /newbot, choose a name and username. Copy the API token provided.

  • 2
    Configure KrustyKlaw

    Set channel = "telegram" in your config and paste the bot token. Optionally restrict to your own Telegram user ID via allowed_users.

  • 3
    Start the bot

    Start KrustyKlaw, then open your bot on Telegram. Send /start and begin chatting.
